How GitHub Sync Works
  • Every time you save your configuration, it's automatically backed up locally
  • If GitHub sync is enabled, the file is also pushed to your repository
  • Your GitHub repository serves as an additional backup and version history
  • Your GitHub token is stored securely and never logged
